1 Что такое условие?

В жизни мы постоянно принимаем решения на основе условий:

  • Если на улице дождь, то я возьму зонт
  • Если я голоден, то я поем
  • Если светофор красный, то я стою

В программировании то же самое! Условный оператор помогает компьютеру принимать решения.

условие_пример.txt 
-- Простое условие
если на_улице_дождь то
    взять зонт
иначе
    зонт не_брать

-- Условие в Minecraft
если перед_агентом_блок то
    повернуть налево
иначе
    двигаться вперёд
💡 Важно: условие — это вопрос, на который можно ответить «да» или «нет». Если ответ «да» — выполняются одни команды, если «нет» — другие (или ничего).

2 Определение ID блока

В Minecraft каждый блок и предмет имеет уникальный номер — ID. Зная ID, мы можем проверять, какой блок находится перед агентом или в его инвентаре.

Название блокаID
Земля (Dirt)3
Камень (Stone)1
Дерево (Oak Wood)17
Изумруд (Emerald)133
Золото (Gold)14
Алмаз (Diamond)56

Как узнать ID блока в игре?

  • В творческом режиме наведи на блок — в подсказке будет ID
  • Используй команду /testforblock — она показывает ID блока
  • В редакторе кода есть блок ID блока — он показывает номер

3 Условный оператор «если ... то»

В Minecraft Education есть специальный блок для условий — «если ... то». Он позволяет агенту проверять что-то и выполнять разные команды.

БлокЧто проверяетПример
если [ID блока] == [значение]Равны ли ID блока и числоесли ID блока == 3 → это земля
если перед агентом блокЕсть ли блок перед агентомесли перед_агентом_блок → повернуть
если [условие] то [команды] иначе [команды]Выполняет разные командыесли ID == 1 то поставить камень иначе поставить землю
условие_агент.txt 
-- Агент проверяет, какой блок перед ним
если ID_блока_перед_агентом == 3 то
    сказать "Это земля!"
    поставить блок_изумруда
иначе
    сказать "Это не земля"
    двигаться вперёд
⚡ Важно: условие может проверять ID блока, наличие блока, количество предметов и многое другое. Это основа для создания умных программ!

5 Практика: Агент-продавец

Теперь создадим настоящую программу! Агент будет работать продавцом в магазине.

Как это работает:

  • Игрок пишет в чате «цены» — агент показывает стоимость товаров
  • Игрок кидает изумруды на землю и пишет «купить [1-5]» — агент собирает их
  • Агент проверяет, какой предмет запросил игрок (число после «купить») и сколько изумрудов дал игрок
  • В зависимости от количества изумрудов агент выдаёт нужный предмет или говорит, что изумрудов не хватает
