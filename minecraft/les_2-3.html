1 Повторяем команды для шахтёра

Для копания тоннеля нам понадобятся команды движения и разрушения блоков:

КомандаДействиеПример
двигаться вперёдШаг вперёддвигаться вперёд 5 раз
разрушить блокЛомает блок перед агентомразрушить блок
подняться вверхПоднимается на блокподняться вверх
опуститься внизОпускается на блокопуститься вниз
повернуть налево/направоПоворот на 90°повернуть налево
поставить блокСтавит блок перед агентомпоставить блок (факел)
⚡ Важно: для копания тоннеля высотой 2 блока агент должен сначала разрушить блок на уровне головы, затем на уровне ног, и только потом двигаться вперёд!

2 Проверяем блок перед агентом

Чтобы агент мог определить, что за блок перед ним, используется команда ID блока в сочетании с условием.

проверка_блока.txt 
-- Проверяем, что за блок перед агентом
если ID_блока_перед_агентом == 1 то  -- 1 = камень
    сказать "Это камень, копаем дальше"
иначе если ID_блока_перед_агентом == 56 то  -- 56 = алмазная руда
    сказать "Я нашёл алмазы! 🎉"
    остановить программу  -- завершаем работу
иначе
    сказать "Что-то странное..."
БлокIDЧто делает агент
Камень (Stone)1Продолжает копать
Угольная руда16Продолжает копать
Железная руда15Продолжает копать
Золотая руда14Останавливается и сообщает
Алмазная руда56Останавливается и сообщает
Изумрудная руда133Останавливается и сообщает
💡 Остановка программы: команда остановить программу (или выйти) завершает выполнение всех команд. Это полезно, когда агент нашёл что-то ценное.

🛠️ Практическое задание: копка тоннеля

Выполнено шагов: 0 из 5

Создай в Minecraft Education программу для автоматического копания тоннеля высотой 2 блока.

Основное задание:

💡 Подсказка: для копания тоннеля используй цикл повторить N раз. Факелы ставь на стену с помощью повернуть налево, поставить блок, повернуть направо.
🌟 Идея: для проверки блока используй условие с ID блока. Если блок не камень (ID != 1), значит агент нашёл руду или что-то ценное!
