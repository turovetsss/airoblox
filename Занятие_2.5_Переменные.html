0 Теория: что такое переменная?

📦 Коробочка с именем

Переменная — это контейнер, в котором можно сохранить какое-то значение (число, текст и т.д.). У каждой переменной есть имя, по которому мы к ней обращаемся.

Представь, что у тебя есть пустая коробка. Ты наклеиваешь на неё этикетку «монеты» и кладёшь туда число 100. Потом в любом месте программы ты можешь сказать: «Дай мне содержимое коробки 'монеты'» и получить 100.

📝 Как объявить переменную в Lua

В Roblox Lua переменные обычно объявляют с ключевым словом local, чтобы они были видны только в текущем скрипте.

Синтаксис:

local имяПеременной = значение

Примеры:

local age = 12
local name = "Бобик"
local isReady = true
local price = 2.5
💡 Правило хорошего тона: всегда используй local, чтобы переменная не «убежала» в другие скрипты.

0 Теория: типы данных в Lua

В переменную можно положить данные разных видов. Вот самые главные:

ТипОписаниеПример
number Числа (целые и дробные) local score = 150
local pi = 3.14
string Текст (строка) local hello = "Привет"
local path = 'game.Workspace'
boolean Логическое значение: true или false local flag = true
local gameOver = false
nil Отсутствие значения («пусто») local nothing = nil
table Таблица — может хранить много значений local colors = {"Red", "Green", "Blue"}

📋 Таблицы — переменные с ячейками

Таблица похожа на список или шкаф с ящиками. В ней можно хранить сразу несколько значений под одним именем.

local fruits = {"apple", "banana", "cherry"}
print(fruits[1]) -- выведет "apple"
print(fruits[2]) -- выведет "banana"

Нумерация начинается с 1 (не с нуля, как в некоторых других языках).

📘 Шпаргалка: переменные и типы данных

📌 Объявление переменной
local myVar = значение — правильное объявление.
Имя переменной: буквы, цифры, символ подчёркивания. Не может начинаться с цифры.
Примеры имён: playerName, _count, box2.
🔢 Числа (number)
local a = 10
local b = 3.14
Можно выполнять арифметику: a + b, a * 2, a / 5.
📝 Строки (string)
local s1 = "Привет"
local s2 = 'Мир'
Склеивание: s1 .. " " .. s2 → "Привет Мир".
✔️ Логические (boolean)
local isActive = true
local isHidden = false
Используются в условиях if.
📋 Таблицы (table)
local t = {10, 20, 30}
Доступ: t[1] → 10, t[2] → 20.
Добавление: table.insert(t, 40).
🌍 Область видимости
Переменная local видна только внутри своего блока/функции.
Если объявить без local, переменная станет глобальной — лучше так не делать.

📏 Задание 5 — Изменение размера Part при клике

Создай кнопку, которая при каждом нажатии увеличивает размер Part на 20%. Используй переменную scale для хранения текущего масштаба.

  • Начальный размер Part: 4 × 2 × 4.
  • При каждом клике умножайте размер на новый масштаб scale = scale + 0.2.
  • Выводи в Output текущий масштаб.
Выполнено шагов: 0 из 3
  • local part = script.Parent
    local click = part:FindFirstChild("ClickDetector")
    local scale = 1

    function onClick()
      scale = scale + 0.2
      part.Size = Vector3.new(4, 2, 4) * scale
      print("Новый масштаб: " .. scale)
    end

    click.MouseClick:Connect(onClick)
💡 Подсказка: Размер умножается на коэффициент scale — чем больше кликов, тем больше Part.
