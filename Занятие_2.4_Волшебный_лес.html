💻 Челлендж 7 — Анимация леса (БОНУС)

⭐ ⭐ ⭐ ⭐ ⭐

Сложность: 🌿🌿🌿🌿 Экспертная ⭐ БОНУС

Добавь скрипт, который оживит лес: свечение, вращение кристаллов и мерцание цветов.

Выполнено шагов: 0 из 4
скрипт_волшебный_лес.lua 
-- Оживляем волшебный лес

local isGlowing = true

while true do
    -- Проходим по всем частям леса
    for i, child in pairs(script.Parent:GetChildren()) do
        if child:IsA("BasePart") then
            -- Меняем материал на Neon (если ещё не Neon)
            if child.Material ~= "Neon" then
                -- Сохраняем оригинальный цвет
                local origColor = child.BrickColor
                child.Material = "Neon"
                child.BrickColor = origColor
            end

            -- Меняем цвет на случайный каждые 2 секунды
            if math.random(1, 20) == 1 then
                child.BrickColor = BrickColor.random()
            end

            -- Вращаем кристаллы (если они есть)
            if child.Name:find("Crystal") then
                child.Rotation = child.Rotation + Vector3.new(0, 0.5, 0)
            end
        end
    end

    -- Пауза 0.1 секунды
    wait(0.1)
end
🧠 Что делает скрипт? Он превращает все части леса в Neon, периодически меняет их цвет на случайный, а кристаллы (если их назвать "Crystal") вращает.
