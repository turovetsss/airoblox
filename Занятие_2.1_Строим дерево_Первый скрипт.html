1 Инструменты — наши руки

Чтобы создать дерево, нужно уверенно владеть четырьмя главными инструментами. Вспомним их:

Инструменты трансформации
Инструмент Горячая клавиша Что делает
Select (Выбор) Q Выделяет объект
Move (Перемещение) W Двигает по осям X, Y, Z
Scale (Масштаб) E Меняет размер
Rotate (Вращение) R Вращает вокруг оси
🖥️

Как открыть свойства объекта в Roblox Studio

  1. Выдели объект – кликни по нему в окне Workspace (3D-вид) или в Explorer (справа).
  2. Открой панель Properties – если её нет, включи через меню View → Properties (или нажми Ctrl+1).
  3. В панели Properties ты увидишь все параметры объекта: Size, Anchored, BrickColor, Material и другие.
🌳 Совет: Чтобы быстро скопировать объект, используй Ctrl+D (дублирование). Это пригодится для веток и листвы!

3 Что такое скрипт?

Скрипт — это программа, написанная на языке Lua, которая говорит игре, что делать. Скрипты оживляют игры: без них объекты просто стоят на месте.

Скрипты в Roblox
Тип скриптаГде находитсяЧто делает
ScriptВнутри объекта (Server)Выполняется на сервере — для игровой логики
LocalScriptВнутри игрока (Client)Выполняется на клиенте — для интерфейса
ModuleScriptВ ReplicatedStorageХранит общий код для других скриптов
📄

Как добавить скрипт в Roblox Studio

  1. Выбери объект (например, твоё дерево) в Explorer (Проводник).
  2. Нажми на + (плюс) рядом с объектом или нажми Ctrl+Shift+S.
  3. Выбери Script — появится новый скрипт внутри объекта.
  4. Дважды кликни по скрипту — откроется редактор кода.
💡 Важно: Скрипт выполняется, только когда объект находится в Workspace. Если объект скрыт или удалён — скрипт не работает.

Основы языка Lua

КодЧто делает
print("Привет!")Выводит текст в консоль (Output)
wait(2)Ждёт 2 секунды
script.ParentОбращается к родительскому объекту (тому, в котором лежит скрипт)
object.Material = "Neon"Меняет материал объекта на Neon
object.BrickColor = BrickColor.new("Bright red")Меняет цвет объекта

💻 Задание 5 — Первый мини-скрипт для дерева

Давай оживим наше дерево! Мы напишем скрипт, который будет вращать дерево и менять цвет листвы.

Выполнено шагов: 0 из 4
скрипт_дерева.lua 
while true do

	for i, child in pairs(script.Parent:GetChildren()) do
		if child:IsA("Part") then
			child.BrickColor = BrickColor.random()
		end
	end

	wait(0.3)
end
🧠 Что делает этот скрипт?
1. while true do — бесконечный цикл (скрипт работает вечно).
2. script.Parent.Rotation — вращает дерево на 1 градус по оси Y каждый раз.
3. for child in pairs(...) — перебирает все части внутри дерева и меняет их цвет на случайный.
4. wait(0.3) — делает паузу 0.3 секунды, чтобы всё не происходило слишком быстро.
