📋 Как пользоваться этим шаблоном

Это заготовка занятия со всеми видами блоков сразу. Открывайте файл в любом текстовом/код-редакторе (подойдёт даже Блокнот, но удобнее — бесплатный VS Code или Notepad++) и меняйте текст между тегами.

  • Каждый блок в коде отмечен комментарием <!-- БЛОК: ... --> — по нему легко найти нужное место.
  • Чтобы добавить ещё один такой же блок (например, ещё один вопрос в опросе или ещё один шаг в задании) — скопируйте весь кусок кода от одного комментария до следующего и вставьте ниже, затем поменяйте текст.
  • Не удаляйте кавычки ", символы < > и слова внутри class="..." — это разметка, на ней держится оформление. Меняйте только обычный текст на русском.
  • Место для фото/видео отмечено пунктирной рамкой — просто замените тег <div class="media-placeholder"> на настоящую картинку: <img src="имя_файла.jpg"> (файл картинки положите рядом с этим html-файлом).
  • В опросе (Quiz) у каждого вопроса есть data-correct="a" — буква должна совпадать с value правильного варианта ответа.
